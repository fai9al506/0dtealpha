# Trading References Library

Drop documents into the appropriate folder. Claude Code reads these on-demand to answer trading questions, improve setup detection, and refine the system.

## Folder Structure

### `volland/`
- User guide, white paper, documentation
- Discord community insights and shared experiences
- Charm/vanna/gamma interpretation guides
- Paradigm explanations

### `gex/`
- Gamma exposure studies and papers
- GEX interpretation frameworks
- Dealer hedging mechanics

### `orderflow/`
- Order flow / delta / CVD studies
- Absorption and exhaustion patterns
- Footprint chart interpretation

### `general/`
- Options Greeks references
- Market microstructure
- 0DTE-specific strategies and research

## Supported Formats
- `.md` (markdown) — best for text, easy to search
- `.txt` (plain text)
- `.pdf` — Claude can read PDFs directly
- `.png` / `.jpg` — Claude can view images/charts

## Tips
- For Discord chats: copy-paste the useful exchanges into a `.md` file with some context
- For long PDFs: add a short summary at the top so Claude knows what's inside
- Name files descriptively: `volland-charm-interpretation.md` > `doc1.md`
