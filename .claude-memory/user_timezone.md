User's laptop is set to KSA (Kingdom of Saudi Arabia) time = UTC+3.
Discord chat exports use laptop local time, so all Discord timestamps are KSA.

**Conversion:** KSA → ET = subtract 7 hours (during US EDT, Mar-Nov).

Example: 14:34 KSA = 07:34 AM ET.
