---
name: Don't suggest wrapping up
description: Never suggest ending the session — the user will say bye when they're done
type: feedback
---

Stop asking "want me to wrap up?" or "shall we end the session?" — the user finds it annoying.

**Why:** The user will say "bye" or "done" when they want to end. Suggesting it repeatedly feels like you're trying to cut the conversation short.

**How to apply:** Never suggest wrapping up. Just keep working until the user explicitly says they're done.
