---
name: Telegram only for live setups
description: Portal/logging setups should NOT send Telegram — avoid confusion. Data is collected in portal trade log.
type: feedback
---

Telegram alerts are ONLY for live (auto-traded) setups. Portal/logging setups collect data silently via the portal trade log — do NOT say "we're collecting zero data" just because Telegram is off. The portal IS the data collection mechanism.

**Why:** User wants clean Telegram channel — only actionable signals. Portal-only setups firing Telegram would create noise and confusion.

**How to apply:** When discussing portal-only setups, never suggest enabling Telegram as a "fix". The data IS being collected through the portal. Focus on analyzing that data instead.
